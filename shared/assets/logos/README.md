# CFES Logos

This directory should contain CFES logo files in PNG format.

## Required Files

- `CFES_Logo.png` - Main CFES logo (high resolution)
- `CFES_Logo_EN.png` - English version with text (optional)
- `CFES_Logo_FR.png` - French version with text (optional)

## Usage

These logos are referenced in the LaTeX documents via:
```latex
\includegraphics{../../shared/assets/logos/CFES_Logo}
```

Note: The `.png` extension is automatically added by LaTeX.
